Vetores e Geometria Analítica

1. Espaço Afim

O espaço afim é uma estrutura onde se definem vetores. A cada par de pontos P, Q ∈ ℝⁿ associa-se um vetor PQ⃗.

Para cada ponto P e cada vetor ⃗v existe um e um só ponto Q tal que ⃗v = PQ⃗.

Relação de Chasles

Dados P, Q, R três pontos, tem-se que:

PQ⃗ + QR⃗ = PR⃗

2. Operações com Vetores

Operação Definição Propriedades
Igualdade u⃗ = (u₁, u₂, u₃) = (v₁, v₂, v₃) ⇔ u₁ = v₁ ∧ u₂ = v₂ ∧ u₃ = v₃ -
Adição u⃗ + ⃗v = (u₁ + v₁, u₂ + v₂, u₃ + v₃) Comutativa, Associativa
Multiplicação por escalar k · u⃗ = (ku₁, ku₂, ku₃) Distributiva
Produto escalar u⃗ · ⃗v = u₁v₁ + u₂v₂ + u₃v₃ Comutativo, Distributivo

Norma e Ângulo entre Vetores

∥u⃗∥ = √(u₁² + u₂² + u₃²)
θ = cos⁻¹(u⃗ · ⃗v / (∥u⃗∥ · ∥⃗v∥))

Vetores ortogonais: u⃗ · ⃗v = 0

Vetores unitários: ∥u⃗∥ = 1

3. Produto Vetorial e Misto

Produto Vetorial

u⃗ × ⃗v = det⎡i⃗ j⃗ k⃗⎤ ⎢u₁ u₂ u₃⎥ ⎣v₁ v₂ v₃⎦

Resulta em um vetor perpendicular a u⃗ e ⃗v. Se colineares, u⃗ × ⃗v = ⃗0.

Produto Misto

u⃗ · (⃗v × w⃗) = det⎡u₁ u₂ u₃⎤ ⎢v₁ v₂ v₃⎥ ⎣w₁ w₂ w₃⎦

Representa o volume do paralelepípedo definido pelos três vetores.

4. Retas no Espaço

Dado um ponto P e um vetor u⃗ não nulo, a reta que passa por P com direção u⃗ é o conjunto:

L(P, u⃗) = {P + ku⃗ | k ∈ ℝ}

4.1 Equações da Reta

Tipo Forma
Vetorial (x, y, z) = (x₀, y₀, z₀) + k(u₁, u₂, u₃)
Paramétricas x = x₀ + ku₁
y = y₀ + ku₂
z = z₀ + ku₃
Cartesianas (x - x₀)/u₁ = (y - y₀)/u₂ = (z - z₀)/u₃

Exercício Resolvido

Verifique se P = (2, -1, 4) pertence à reta L(Q, ⃗v) com Q = (-3, 1, 1) e ⃗v = (1, -2, 3).

Solução: O sistema resultante é inconsistente, portanto P não pertence à reta.

5. Planos no Espaço

Dado um ponto P e dois vetores u⃗, ⃗v linearmente independentes, o plano que contém P gerado por u⃗ e ⃗v é:

α = {P + su⃗ + t⃗v | s, t ∈ ℝ}

5.1 Equações do Plano

Tipo Forma
Vetorial (x, y, z) = (x₀, y₀, z₀) + s(u₁, u₂, u₃) + t(v₁, v₂, v₃)
Paramétricas x = x₀ + su₁ + tv₁
y = y₀ + su₂ + tv₂
z = z₀ + su₃ + tv₃
Cartesiana a(x - x₀) + b(y - y₀) + c(z - z₀) = 0
onde (a, b, c) = u⃗ × ⃗v

Exercício Resolvido

Verifique se o ponto Q = (2, 0, 3) pertence ao plano definido por P = (1, 1, 1), u⃗ = (2, -1, 3), ⃗v = (-1, 0, 2).

Solução: O sistema não tem solução, portanto Q não pertence ao plano.

6. Distâncias

6.1 Distância de Ponto a Reta

Para L(P, u⃗) e Q ∉ L:

  1. Encontre I ∈ L tal que QI⃗ ⊥ u⃗
  2. Calcule ∥QI⃗∥

6.2 Distância entre Retas

Para retas r₁ e r₂:

  • Concorrentes: d(r₁, r₂) = 0
  • Paralelas: d(r₁, r₂) = d(P ∈ r₁, r₂)
  • Reversas: Use plano que contém r₂ e é paralelo a r₁

Exercício Resolvido

Calcule a distância entre as retas r₁: (1,2,0) + k(-1,3,2) e r₂: {y=1-3x, z=-1-2x}.

Solução: As retas são paralelas e a distância é √91/7.

7. Superfícies de Revolução

Uma superfície de revolução é gerada pela rotação de uma curva (geratriz) em torno de um eixo fixo.

Exemplos:

  • Reta paralela ao eixo Oz → Cilindro circular
  • Reta oblíqua ao eixo Oz → Cone
  • Reta inclinada em relação a eixo não paralelo → Superfície complexa
